# Weather-Chatbot
Weather Chatbot in NodeJS

Clone repository and run app.js in your cmd.

## Test Case 

Examples: 

User Input:
`Hello`

Output:
`Hi there !`
`Enter a city name to know what's the weather like there : `

User Input: 
`Paris`

Output:
`Right now, it is quite warm in Paris. It is Sunny and 26 degrees Celsus.`
